# 🛒 E-Commerce Multi-Actor System 

E-commerce platform with role-based access for Customers, Sellers, and Admins.



## 👥 Team

| Member | Role | Responsibilities |
|--------|------|-----------------|
|   | Full-Stack | Authentication + Admin Panel |
|   | Frontend Lead | Home Page + Design System |
|   | Frontend | Products Catalog |
|   | Frontend | Shopping Cart + Checkout |
|   | Full-Stack | Customer & Seller Dashboards |

## 🚀 Features

### For Customers
- ✅ Browse products
- ✅ Search & filter
- ✅ Shopping cart
- ✅ Checkout process
- ✅ Order history

### For Sellers
- ✅ Manage products (CRUD)
- ✅ Process orders
- ✅ Sales analytics

### For Admins
- ✅ User management
- ✅ Product moderation
- ✅ Platform analytics

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5.3
- Chart.js
- LocalStorage API
